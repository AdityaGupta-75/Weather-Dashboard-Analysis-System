# 🌦️ Weather Dashboard Analysis System

## 📌 Project Description
This project is a Weather Dashboard built using Python and Open-Meteo API. 
It displays historical weather data including temperature, wind speed, and weather conditions in calendar and graph format.

## 🚀 Features
- 📅 Weather Calendar View
- 🌡️ Temperature Graph (Max & Min)
- 🌬️ Wind Speed Graph
- 🏙️ City-based Weather Search
- 📊 Interactive Dashboard using Widgets

## 🛠️ Technologies Used
- Python
- Requests (API)
- Matplotlib
- IPyWidgets
- Open-Meteo API
- Google Colab

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Install libraries using requirements.txt
3. Run all cells
4. Enter city name and select month
5. View Calendar or Graph
